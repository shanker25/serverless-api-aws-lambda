## Project Scope

This project demonstrates the development of a serverless REST API using AWS Lambda and API Gateway. The API provides multiple endpoints that return student information, skills, projects, and contact details in JSON format.

## Technologies Used

* AWS Lambda
* Amazon API Gateway
* Python
* GitHub

## Architecture

User → API Gateway → AWS Lambda → JSON Response

## API Endpoints

### GET /student

Returns student information.

### GET /skills

Returns technical skills.

### GET /projects

Returns project details.

### GET /contact

Returns contact information.
